# LTS Prototipo

Prototipo en Blazor para ejemplo de Latis Web

## LTS_Proto.DB

Proyecto de Base de Datos. Contiene la estructura de la base de datos. Con el comando __Publish__ (botón derecho sobre el proyecto DB, publish) se crea la base de datos usando el "String de conexión adecuado".

## LTS_Proto.Core

Proyecto que almacena las clases que representan los Modelos (Tablas) y View que usuará la aplicación.

## LTS_Proto.BL

Proyecto para "Bussiness Logic". Contiene los BO.

## LTS_Proto.Web

Proyecto Web (En Blazor) Prototipo
